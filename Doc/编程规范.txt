一、命名规范
    1. 类型名
        首字母大写，不加任何前缀；
        接口类型使用I开头；

    2. 成员变量
        class的成员变量使用m_前缀，若是指针，则m_后接p类型前缀，句柄则使用h前缀，bool类型使用b前缀，其他类型均不加类型前缀；
        struct的成员变量不使用m_前缀，首字母小写；
        若没有合适的类型前缀，则直接m_后接首字母大写即可；

    3. 全局变量
        全局变量使用g_前缀；
        全局常量使用全大写；

    4. 函数
        函数首字母一律大写，使用驼峰命名法；
        函数形参一律小写开头，使用驼峰命名法；
        函数中的局部变量一律小写开头，使用驼峰命名法，有特殊含义的常用术语或者缩写可以使用全大写；
        函数体的大括号一律换行，如下所示：
        void A()
        {
            return;
        }
